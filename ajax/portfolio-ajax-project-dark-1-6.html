Budget Analysis System

Project Info:

• A device built with a combination of 4 sensors and a Android tablet capable of fixed-axis 360 degree rotation to face users based on touch sensor input or automated with sonar sensors.

• Worked on Android application for user and built website for order management.

• User-side: Android Application, Arduino Uno, Nema-17 stepper motor, TTP223 touch sensor, HC-05 Bluetooth module.

• Management-side: Bootstrap, JavaScript, HTML5, CSS3, Firebase.
